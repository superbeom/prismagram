# Prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [O] Create account
- [O] Request Secret
- [O] Confirm Secret (Login)
- [O] Like / Unlike a photo
- [O] Comment on a photo
- [O] Search by user
- [O] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit / Delete the photo
- [ ] See the feed
