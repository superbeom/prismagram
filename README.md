# Prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [O] Create account
- [O] Request Secret
- [O] Confirm Secret (Login)
- [O] Like / Unlike a photo
- [O] Comment on a photo
- [O] Search by user
- [O] Search by location
- [O] Follow User
- [O] Unfollow User
- [ ] Edit my profile
- [ ] See user profile
- [ ] See the full photo
- [ ] Upload a photo
- [ ] Edit / Delete the photo
- [ ] See the feed
