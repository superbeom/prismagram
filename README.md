# Prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [O] Create account
- [ ] Log in
- [ ] Upload a photo
- [ ] Edit / Delete the photo
- [ ] See the feed
